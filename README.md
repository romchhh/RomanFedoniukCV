# Roman Fedoniuk - CV Website

Сучасний веб-сайт резюме з підтримкою трьох мов (українська, англійська, польська).

## Особливості

- 🌍 **Мультимовність**: Підтримка української, англійської та польської мов
- 📱 **Адаптивний дизайн**: Оптимізовано для всіх пристроїв
- ⚡ **Швидкість**: Побудовано на Vite + React
- 🎨 **Сучасний дизайн**: Мінімалістичний стиль з Tailwind CSS
- 🔗 **Інтерактивні елементи**: Клікабельні контакти та посилання

## Технології

- **React 18** - Основний фреймворк
- **Vite** - Швидкий бандлер та dev сервер
- **Tailwind CSS** - Утилітарний CSS фреймворк
- **Lucide React** - Іконки
- **PostCSS** - Обробка CSS

## Встановлення та запуск

1. **Встановіть залежності:**
   ```bash
   npm install
   ```

2. **Запустіть dev сервер:**
   ```bash
   npm run dev
   ```

3. **Відкрийте браузер:**
   Сайт буде доступний за адресою `http://localhost:3000`

## Доступні команди

- `npm run dev` - Запуск dev сервера
- `npm run build` - Збірка для продакшену
- `npm run preview` - Попередній перегляд збірки

## Структура проекту

```
src/
```

## Налаштування

### Додавання фото

В компоненті `Resume.jsx` є поле для введення URL фото. Просто введіть посилання на ваше зображення в поле "Photo URL".

### Редагування контенту

Весь текстовий контент знаходиться в об'єкті `content` в компоненті `Resume.jsx`. Ви можете легко редагувати:

- Особисту інформацію
- Досвід роботи
- Проєкти
- Навички
- Освіту
- Контакти

### Зміна стилів

Стилі налаштовані через Tailwind CSS класи. Основні кольори:
- Чорний: `bg-black`, `text-black`, `border-black`
- Білий: `bg-white`, `text-white`
- Сірий: `bg-gray-100`, `text-gray-600`

## Деплой

Для деплою на статичний хостинг:

1. Зберіть проект:
   ```bash
   npm run build
   ```

2. Завантажте папку `dist` на ваш хостинг
